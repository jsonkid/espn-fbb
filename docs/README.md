# Documentation Index

This directory is organized by audience and use case.

## User Docs

- `COMMANDS.md`: CLI command syntax, flags, examples, and exit codes.
- `OUTPUT_SCHEMA.md`: JSON output contract for each command.

## Developer Docs

- `ARCHITECTURE.md`: module responsibilities and runtime flow.
- `API_INTEGRATION.md`: ESPN endpoint usage, auth model, and payload quirks.
- `ANALYTICS_METHOD.md`: deterministic analytics and projection logic.
- `CACHE_AND_PERFORMANCE.md`: cache keys/TTLs and request budget behavior.
- `TESTING.md`: test strategy and how to run tests.

## Project Tracking

- `CHANGELOG.md`: notable user-facing and schema changes.
- `ROADMAP.md`: open questions and future enhancements.

# Roadmap

## Near-Term

- Tighten category projection precision policy (documented rounding guarantees).
- Improve outlook confidence signals for sparse-stat cases.

## Medium-Term

- Optional machine-readable schema export (JSON Schema) for downstream validators.
- More explicit versioned compatibility notes for schema evolution.
- Broader fixtures for league format differences (`matchupPeriodLength != 1`).

## Open Questions

- Best default behavior when future matchup rows are missing opposing team context.
- Whether to expose optional debug payload sections in a stable contract.
- Whether lineup swap heuristics should become separately tunable.
